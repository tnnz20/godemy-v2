## Penjelasan Kode Program


### 1. Penggunaan Package

Dalam bahasa pemrograman Go, `package` adalah kumpulan kode yang di 
kelompokkan berdasarkan fungsi nya. Package dapat digunakan 
untuk membagi kode menjadi bagian-bagian yang lebih kecil dan lebih 
mudah dikelola. `Package` juga dapat digunakan untuk berbagi kode 
dengan program lain.

Package dapat dibagi menjadi dua jenis, yaitu:
- Package bawaan
- Package pihak ketiga

**Package bawaan** adalah package yang sudah disediakan oleh bahasa 
pemrograman Go. Package bawaan biasanya berisi fungsi dan variabel 
yang umum digunakan, seperti fungsi `fmt.Println()`.

**Package pihak ketiga** adalah package yang dibuat oleh pihak lain. 
Package pihak ketiga biasanya berisi fungsi dan variabel yang lebih 
spesifik, seperti fungsi untuk mengakses database atau fungsi untuk 
melakukan analisis data.

Setiap program Go harus memiliki `package`. Setiap project harus ada
minimal satu file dengan nama package **main**. File yang ber-package
main, akan di eksekusi pertama kali ketika program di jalankan.

Berikut adalah sintak penulisan `package`.

```go 
package <nama-package>
package main
```

### 2. Penggunaan Import

Keyword `import` digunakan untuk meng-import atau memasukan
library lain ke dalam file program yang kita buat, dengan menggunakan keyword
`import` ini kita bisa memanggil sebuah function yang dibuat di file lain
maupun function yang dibuat oleh orang lain.
library `fmt` merupakan salah satu library bawaan yang disediakan oleh
Go, isinya banyak fungsi untuk keperluan _I/O_ yang berhubungan
dengan text.

Berikut adalah sintak penulisan keyword import:

```go 
import <”nama-package”>
import "fmt"
```

### 3. Penggunaan fungsi main

Dalam bahasa pemrograman Go, fungsi `main()` memiliki peran yang 
sangat khusus dan *esensial*. Ia merupakan titik awal eksekusi program 
Go. Berikut adalah penjelasan tentang fungsi `main()`:

-  Ketika sebuah program Go dijalankan, compiler Go akan 
   mencari fungsi `main()`.
-  Jika fungsi `main()` tidak ditemukan, compiler akan menampilkan 
   pesan kesalahan.
-  Jika fungsi `main()` ditemukan, compiler akan memulai eksekusi 
   program dari fungsi tersebut.

Dalam sebuah proyek Go harus ada 1 file program yang di dalamnya 
berisi sebuah fungsi bernama main. Fungsi main akan dipanggil 
pertama kali pada saat eksekusi program
   
```go showLineNumbers {1,3}
func main( ){
   <isi-fungsi>
}
```

### 4. Penggunaan fmt

Fungsi `fmt.Println()` digunakan untuk menampilkan text ke layar (pada
konteks ini, terminal atau console). Di program pertama yang telah buat, 
fungsi ini menampilkan tulisan **Hello world**.
Sintak penulisan keyword `fmt.Println()` bisa dilihat pada contoh berikut:

```go
fmt.Println(<isi-parameter>)
fmt.Println("Hello World")
```

Fungsi `fmt.Println()` berada dalam library `fmt`, jadi untuk
menggunakannya diperlukan library tersebut untuk di-import terlebih
dahulu.

Fungsi `fmt.Println()` dapat menampung parameter yang tidak terbatas
jumlahnya. Semua data parameter akan dimunculkan kedalam console.
