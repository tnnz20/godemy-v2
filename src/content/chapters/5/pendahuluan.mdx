“Pada materi sebelumnya kita sudah berkenalan dengan tipe data, control – flow, 
serta penggunaan looping. Pada materi ini kita akan mempelajari sebuah tipe data 
yang kompleks yaitu tipe data komposit. Tipe data komposit memiliki hubungan 
dengan tipe data yang kita pelajari sebelumnya. Tipe data komposit adalah salah 
satu konsep yang penting dalam pemrograman, yang memungkinkan kita untuk 
menggabungkan beberapa tipe data dasar ke dalam struktur yang lebih kompleks. 
Dalam materi ini, kita akan memahami bagaimana tipe data komposit, seperti array, 
slice, map, dan struct, memungkinkan kita untuk mengorganisasi, menyimpan, dan 
memanipulasi data dengan lebih efisien dalam program Go.”


Pada akhir sub materi ini, Anda diharapkan dapat:

- Dapat memahami dan menggunakan tipe data komposit array Go
- Dapat memahami dan menggunakan tipe data komposit slice Go
- Dapat memahami dan menggunakan tipe data komposit map Go
- Dapat memahami dan menggunakan tipe data komposit struct Go
